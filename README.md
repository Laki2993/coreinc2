# Core
Веб-приложение для создания и управления задачами.
## Функционал
- Регистрация и аутентификация (Laravel Breeze)
- Восстановление пароля
- Загрузка аватара профиля
- Создание проекта
- Удаление  проекта
- Присоединение к проекту по уникальному идентификатору проекта
- Личный кабинет
- Создание задач
- редактирование задач
- Удаление задач
- Отправка сообщений
## Технологии
- Php 8.1+
- Laravel 10+
- SupaBase(PostgreSQL)
- Laravel Breeze
- JavaScript
- CSS
- Laravel Storage (public диск + симлинк)
## Установка
# Кланирование репозитория
git clone https://github.com/Laki2993/coreinc2.git
cd Core
# создайте env файл и вставте в него содержимое файла env.txt
# Установка зависимостей PHP
composer install
# Установка зависимостей Node.js
npm install
# Выполнение миграцийs
php artisan migrate

# Создание симлинка для публичного доступа к файлам (аватары, фото проектов)
php artisan storage:link

# Сборка фронтенд-ассетов 
npm run build

# Запуск локального сервера
php artisan serve
